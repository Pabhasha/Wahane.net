Toyota Corolla Runx

Toyota Corolla Run X is a daily driver which is powered by a performance engine. With a 1800cc engine, this car produces over 190 horsepower and can go up to 180 KPH. This car has the same engine as the Toyota Celica. This car has an aftermarket exhaust and there are only 6 cars in Sri Lanka. There are 3 main types of Toyota Corolla Run X and they are T, R and Z. Out of those 3, this car is type Z. This car is manufactured in 2001.
